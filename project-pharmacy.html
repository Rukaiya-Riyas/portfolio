SDGP Group Project (2025–Present)

HealixPharm

AI-Enhanced Pharmacy Management System designed to automate pharmacy operations including medicine ordering, doctor booking, WhatsApp chatbot communication, and admin management.

Overview

A full-stack healthcare platform that integrates web application, WhatsApp chatbot, and admin system into a single ecosystem for pharmacies and patients.

My Role

UI/UX Design

- Designed high-fidelity UI in Figma
- Focused on healthcare usability & clarity

Backend Development
- Built FastAPI backend for appointment system
- Integrated WhatsApp bot using Twilio API
- Developed REST APIs for booking, payments, admin

Frontend Development
- Built React + Next.js booking system
- Created landing page & reusable components
- Styled using Tailwind CSS

Testing
- API testing & debugging
- Frontend-backend integration testing

Key Features

Doctor Appointment Booking System
WhatsApp Chatbot for medicine ordering
Admin Dashboard for management
Payment workflow integration
Real-time API communication

System Architecture

Built using a microservice-inspired architecture with FastAPI backend, React frontend, and integrated third-party services like Twilio for messaging automation.
