Overview

What is HealixPharm?

HealixPharm automates pharmacy operations end-to-end — from medicine ordering via WhatsApp to doctor appointment booking, admin management, and payment workflows. Built as a group Software Development Group Project (SDGP) at IIT.

The system connects patients, doctors, and pharmacy staff through a unified platform, reducing friction and manual coordination at every step.

My Role

01 — UI/UX Design
Created wireframes for the full system flow. Designed high-fidelity UI in Figma with a focus on healthcare usability and clarity across all user touchpoints.
02 — Backend Development
Built the FastAPI backend for the appointment system. Integrated WhatsApp automation via Twilio API. Developed REST APIs for booking, payments, and admin flows.
03 — Frontend Development
Built the React + Next.js booking portal and landing page with reusable components. Styled using Tailwind CSS for consistent, responsive layouts.
04 — Testing
Conducted API testing and debugging. Handled frontend-backend integration testing to ensure seamless data flow across all system modules.
