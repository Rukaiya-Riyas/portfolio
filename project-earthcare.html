UX/UI Case Study · Figma Prototype

EarthCare

Sustainability App Design

A high-fidelity mobile app prototype that encourages eco-friendly habits through recycling tracking, volunteering, and community engagement — aligned with UN SDGs 11 & 12.

View Figma Prototype →
Overview

The Problem

Many people want to live sustainably but lack a single platform that combines recycling guidance, volunteering opportunities, and ongoing motivation. EarthCare addresses this by unifying these needs into a minimal, accessible mobile experience.

Target Users
Students and young adults · Environmentally conscious individuals · Beginners in sustainability practices
Deliverable
High-fidelity Figma prototype with user-tested flows, accessibility-checked colour system, and iterated navigation.
Design Process

UX Approach

Cognitive Load
Minimal UI
Reduced decision fatigue through clear hierarchy, familiar navigation patterns, and progressive disclosure.
Accessibility
Colour Contrast
All colour pairings tested against WCAG AA standards. Clear system feedback messages for every action.
Research
User Testing
Questionnaire-based testing confirmed ease of use. Resulted in added search functionality and simplified registration flow.
Features

What EarthCare Does

Core Features
Recycling tracker with activity logs
Volunteer event browsing & registration
Eco knowledge hub
Gamified leaderboard system
User profile & progress tracking
Iterations Made
Improved navigation clarity after testing
Replaced unclear filters with a search system
Simplified volunteer registration flow
Enhanced feedback messaging throughout

What Was Achieved

Result
A clean, intuitive, and accessible sustainability app prototype that promotes eco-friendly behaviour through simple, engaging interactions — validated through real user testing and iterated to address feedback.
