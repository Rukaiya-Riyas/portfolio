UX/UI Case Study · Figma Prototype

EarthCare

A sustainability-focused mobile application designed to encourage eco-friendly habits through recycling, volunteering, and community engagement.

Overview

EarthCare is a high-fidelity mobile app prototype built in Figma that promotes sustainable living through simple, engaging, and accessible user experiences aligned with UN SDGs 11 & 12.

Problem Statement

Many users want to live sustainably but lack a simple platform that combines recycling guidance, volunteering, and motivation in one place.

Project Goals

✔ Simplify recycling tracking and volunteering registration
✔ Improve user engagement through feedback systems
✔ Reduce cognitive load with minimal UI
✔ Ensure accessibility for all users

Target Users

Students and young adults
Environmentally conscious individuals
Beginners in sustainability practices

Key Features

Recycling tracker with logs
Volunteer event browsing & registration
Eco knowledge hub
Gamified leaderboard system
User profile & progress tracking

UX / UI Approach

Designed using:

• Cognitive load reduction (minimal UI)
• Consistent navigation patterns
• Accessibility-focused colour contrast
• Clear system feedback messages
• Familiar mobile navigation structure

Usability Testing

Conducted questionnaire-based testing with users.

✔ Easy to use (majority feedback)
✔ Clear system feedback understood by all users
✔ Improvements suggested: search + filtering

Iterations & Improvements

✔ Improved navigation clarity
✔ Added search instead of unclear filters
✔ Simplified volunteer registration flow
✔ Enhanced feedback messaging system

Tools Used

Figma • UX Research • Accessibility Testing Tools

Outcome

A clean, intuitive, and accessible sustainability app prototype that promotes eco-friendly behaviour through simple and engaging interactions.
